# Student Management System (Java + JDBC + MySQL)

## 📌 Project Overview
Console-based Student Management System built using:
- Java
- JDBC (Java Database Connectivity)
- MySQL
- SQL JOIN operations
- Aggregate functions (COUNT, AVG)

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student
- Count total students
- Average marks calculation

## 🛠 Technologies Used
- Java
- JDBC
- MySQL
- Git & GitHub

## 🧠 Concepts Practiced
- CRUD Operations
- SQL Queries
- PreparedStatement
- Exception Handling
- JDBC Connection Handling
- Menu-driven console program

## 📂 Project Structure
- Main.java
- lib/mysql-connector
- MySQL database (studentdb)
